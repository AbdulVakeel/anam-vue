# AnamUI 

A lightweight, AnamUI dashboard built with **Vue 3 + Vite + Tailwind CSS**.

## 🚀 Quick start 

```bash
# 1) Extract and install deps
npm install

# 2) Run dev server
npm run dev
```

If you see errors about charts, run:
```bash
npm install vue3-apexcharts @heroicons/vue
```

## 📦 Tech
- Vue 3, Vite
- Tailwind CSS (dark mode via `class`)
- Vue Router
- ApexCharts (via `vue3-apexcharts`)
- Heroicons

## 🧱 Structure
- `layouts/DefaultLayout.vue` – Sidebar + Topbar shell
- `pages/` – Dashboard, Users, Settings etc.
- `components/` – Cards, Chart, etc.

Enjoy!
